# Audio Files

This folder is intended to store the input audio files used in the experiments of the thesis **“Comparative Study of Audio Signal Compression Algorithms”**.

The experiments in this repository are designed to process `.wav` audio files as input.

## Usage

Place the required `.wav` files in this folder before running the MATLAB scripts.

If the original dataset used in the thesis is not included in the repository, users may provide their own `.wav` files, as long as they update the input paths in the scripts or configuration file accordingly.

## Notes

- Supported input format: `.wav`
- The audio files should preferably be uncompressed PCM WAV files
- Different files may produce different metric values depending on their duration, content, sampling rate, and bit depth
